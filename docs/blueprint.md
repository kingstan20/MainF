# **App Name**: HackMate

## Core Features:

- Authentication: Secure user registration, login, and session management using localStorage.
- Main Feed: Responsive grid layout displaying various post types with view counts and reaction buttons.
- Modal-Based Posting System: Create posts for hackathons, team building, project collaboration, and achievement showcases via a modal interface.
- Profile Page: Professional profile display with editable information, collaboration stats, and hackathon history.
- Settings Page: Account control center for password changes, privacy settings, and theme toggling.
- Data Persistence: Utilize localStorage for storing users, posts, session data, and settings to persist data.
- Initial seed of posts: Creation and addition of posts to localStorage to create a vibrant community on initial load.

## Style Guidelines:

- Background: Black (#000000) to provide a dark, modern canvas.
- Primary: Neon Green (#39FF14) to serve as a vibrant highlight color, embodying a cyber/hacker aesthetic.
- Accent: Soft Green Glow (#6AFF53), used as a subtle effect on borders and elements to add depth.
- Body and headline font: 'Space Grotesk', a sans-serif font providing a modern and tech-focused look, suitable for both headings and body text.
- Custom neon green icons representing various post types (hackathons, collaboration, achievements) to add visual flair.
- Responsive grid-based layout for posts to ensure consistent presentation across devices.
- Subtle hover animations on buttons and interactive elements to enhance user experience and interactivity.
